Privacy policy

FinSight stores your transactions and chat history so the advisor can answer questions about your finances. Data is scoped per user and never shared with other users.

  • You can export all data as JSON from Settings.
  • You can permanently delete your account and all associated data.
  • Bank credentials are never stored — only transaction data you authorize via Plaid.
